# Estatistica descritiva

- Por que estatística existe
- Aprender a organizar dados


## Exemplos de onde usar estatística:

- Conhecendo a altura de uma pessoa, podemos dizer se ela é alta ou baixa para sua idade?
- E se for a altura de um aluno, podemos dizer se ele é alto ou baixo para sua turma?
    - Como descobrir isso sem medir todos os alunos da turma?

## Conceitos-chave:

- Estatística Descritiva : contar uma história com números. Não é adivinhação; é organização inteligente de dados.
- População : todos os alunos da turma (30 pessoas)
- Amostra : um grupo menor (se medíssemos só 10 alunos)
- Dado : cada medida em si (altura de um aluno em particular)
- Rol : lista de dados em ordem crescente


## Prática

Liste as notas de 10 alunos em uma prova (invente números reais):

Notas brutas (desordenadas):

```
7, 9, 5, 8, 6, 9, 7, 8, 10, 5
```

Rol:

```
5, 5, 6, 7, 7, 8, 8, 9, 9, 10
```

Note que, olhando para o rol, conseguimos ver logo que:

- Ninguém tirou menos de 5
- Mais alunos ficaram com 7, 8, 9 do que com 5, 6
- Tem gente que tirou 10


Se você tivesse a lista desordenada tudo ficaria mais confuso.


## Exercício 1 (papel e lápis):

Dados brutos (peso de 12 alunos em kg):

```
62, 58, 71, 65, 58, 69, 71, 63, 67, 60, 71, 65
```

### Tarefa: Organize em rol (crescente) e responda:

- Qual é o peso mínimo?
- Qual é o peso máximo?
- Quantos alunos pesam menos de 65kg?
